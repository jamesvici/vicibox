Networking
==========

By default ViciBox will attempt to use DHCP on the first network interface. Any IP obtained will be shown on the server's display.

.. list-table:: Network Definitions
   :widths: 20 80
   :header-rows: 1
   :align: center

   * - Term
     - Definition
   * - WAN
     - Wire Area Network, the Internet
   * - LAN
     - Local Area Network, private network
   * - ISP
     - Internet Service Provider
   * - NAT
     - Network Address Translation, the firewall
