==============
Specifications
==============

**Minimum** 
    * Four CPU Cores 2.0+Ghz
    * 8-GB RAM
    * 160-GB Drive

**Recommended Telephony and Web**
    * Six CPU Cores 2.0+Ghz
    * 16-GB ECC RAM
    * 500-GB RAID1 SSD

**Dedicated Database 150-Agents**
    * Eight CPU Cores 2.0+Ghz
    * 32-GB ECC RAM
    * 500-GB RAID1 SSD

**Dedicated Database 300-Agents**
    * Sixteen CPU Cores 3.0+Ghz
    * 64-GB ECC RAM
    * 500-GB RAID1 NVMe
